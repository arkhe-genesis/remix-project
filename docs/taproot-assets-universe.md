# Taproot Assets Universe Model

## Universe Federation
The **Universe** is a repository for asset data and proofs.

### Proof Types
- **Genesis Proof**: Proof of initial asset minting.
- **Transfer Proofs**: Chain of transfers.
- **Merkle Proofs**: Inclusion in a Bitcoin UTXO.

### Synchronization
`tapd` nodes can be configured to synchronize their Universe states to provide broader access to proofs across the network, facilitating easier decentralized verifications.
